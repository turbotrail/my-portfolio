# Projects

## 🚀 Featured Project: SpaceWx Deck (AI-powered Space Weather Command Center)
Built a **production-grade, full-stack space weather intelligence platform** that aggregates, analyzes, and visualizes real-time cosmic and environmental data.

This project combines **AI, real-time data pipelines, scientific telemetry, and hardware integration** into a single unified dashboard — designed as a citizen-accessible command center for space weather awareness.

### 🌐 Key Capabilities
- **Multi-Agency Alerts:** Real-time ingestion of space weather alerts, watches, and warnings from NOAA with AI-powered parsing and summarization.
- **Scientific Telemetry:** Live visualization of Solar Wind, X-Ray Flux, and geomagnetic activity with continuous data streams.
- **AI Interpretations:** LLM-driven analysis of complex orbital and space weather data to generate clear, actionable insights and threat assessments.
- **Ancient + Modern Astronomy Fusion:** Mapping traditional systems (Tithi & Nakshatra) with precise orbital mechanics.
- **Local Environmental Sensors:** Integration of real-time indoor air quality, thermal, and atmospheric observatory data.
- **News & Imagery:** Curated feed of space exploration updates and visual datasets.

### ⚙️ Tech Highlights
- Nextjs + FastAPI full-stack architecture
- Real-time API integrations (NOAA and scientific datasets)
- LLM-based data interpretation layer
- Sensor + IoT data ingestion
